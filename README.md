# Data-Analyses-Dashboard
This Project aims at creating a data visualization and analyses dashboard that takes in as input a dataframe, from which it extracts and identifies different variables and offers different types of interactive graphics to visualize it.
### Key Features
- Automatic Graph suggestion
- Adaptability to any data
- Automatic calculation of central tendency and dispersion parameters
- Possibility to group numerical data into categorical features
- Interactive Graphs
- Exportable Graphs and Tables
